# User Behavior & Sales Analysis

## Project Overview
This project analyzes e-commerce transaction data to understand user purchasing behavior and sales performance.

The project includes:
- Data cleaning using Python (pandas)
- User-order relationship reconstruction
- KPI construction (GMV, AOV, purchase frequency)
- SQL analysis
- Power BI dashboard visualization

## Tech Stack
- Python (pandas, numpy)
- SQL
- Power BI

## Project Structure
- data/
- notebooks/
- sql/
- powerbi/
- images/

## Key Insights
The analysis provides insights into sales trends, category performance, and user purchasing behavior.
